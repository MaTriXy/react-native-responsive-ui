# react-native-responsive-ui

Responsive UIs for React Native

## Installation

```sh
npm install react-native-responsive-ui
```

## Usage

```js
import ResponsiveUi from "react-native-responsive-ui";

// ...

const result = await ResponsiveUi.multiply(3, 7);
```
